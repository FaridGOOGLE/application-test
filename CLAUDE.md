# Page d'inscription - Bootcamp IA

## Contexte
Ce projet est une landing page d'inscription à un bootcamp sur l'intelligence artificielle, organisée par STUDERIA (organisme de formation certifié Qualiopi).

## Informations de l'événement
- **Nom** : "Bootcamp Claude - Passez de l'usage curieux à l'usage productif"
- **Date** : Samedi 10 mai 2026
- **Horaires** : 10h00-18h00
- **Tarif** : 490 € HT (590 € TTC)
- **Places limitées** : 100 participants
- **Public** : Professionnels tous secteurs, débutants
- **Formateur** : Faouzi Fieve

## Règles techniques
- Un SEUL fichier `index.html` auto-suffisant
- Tailwind CSS via CDN
- JavaScript inline
- Responsive mobile-first
- Pas de framework (pas de React, pas de Vue)
- Les données du formulaire sont stockées dans localStorage (pas de backend pour le prototype)

## Règles de design
- **Palette** : violet `#534AB7` (principal), gris anthracite `#1F2A44` (texte), orange `#F97316` (CTA/accents), fond blanc cassé `#FAFAFA`
- **Typographie** : Inter via Google Fonts CDN
- **Style** : éditorial sobre, professionnel — inspiration Linear, Stripe, Vercel
- Design aéré, professionnel, style Stripe/Linear
- Sections bien séparées avec beaucoup de respiration
- Mobile-friendly obligatoire

## Règles de contenu
- Français uniquement
- Ton direct, professionnel, pas commercial
- Pas de superlatifs ("révolutionnaire", "game-changer")
- Les témoignages sont fictifs mais réalistes
